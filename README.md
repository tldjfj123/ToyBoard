# 게시판-backend

## 게시판 Back-end 소개

- 제가 가진 백엔드 역량을 보여줄 만한 프로젝트가 없다는 피드백을 받은 후 시작하게 되었습니다.
- 근본인 게시판부터 출발하여 갖다붙일 수 있는건 모두 다 갖다붙일 에정입니다.
- 앞단은 Thymeleaf 로 간단하게 구성 할 예정입니다.
- 배포까지 해보는 걸 목표로 삼고 있습니다.
